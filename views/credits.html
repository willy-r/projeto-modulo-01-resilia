Créditos

História totalmente fictícia retirada das profundezas da mente de William Rodrigues, baseada nos filmes da franquia Invocação do Mal.

Todas as imagens/vídeos/áudios são retirados dos filmes, então todos os direitos vão para a New Line Cinema.

Site construído para fins didáticos, para o final do módulo 01 do curso de desenvolvimento web da Resilia, utilizando somente HTML, CSS e JavaScript.
